## 4. 적용된 개선 사항 (A+B+F+C)

### A. LAW_BOOST 26개 리더 확대 (13개 신규 추가)
기존 13개 + 신규 13개:
- L05(의료), L06(손해배상), L09(건설), L16(보험), L18(금융)
- L20(조세), L22(형사), L26(지식재산), L31(행정), L37(환경)
- L42(저작권), L45(소비자), L50(엔터)

각 리더별 핵심 법률 조문 7~8개를 DRF 검증 통과 가능한 형태로 추가.

### B. 구조 지시 프롬프트 강화
Gemini에게 "반드시 마지막에 '## 법률 근거' 섹션을 작성하고, 법령명+조문번호를 최소 3개 이상 나열" 지시 추가.

### F. 법률 근거 자동 부착 (Stage 3.9)
Gemini가 "법률 근거" 섹션을 누락한 경우, LAW_BOOST에서 응답에 인용된 법률을 자동 추출하여 `## 법률 근거` 섹션을 부착. DRF 검증 전에 실행되어 검증 통과 가능한 법조문만 포함.

### C. 최소 길이 재생성 로직 강화
- 재생성 시 "N자 이상 작성" 명시적 길이 지시 추가
- 1회 → 최대 2회 재시도로 확대
- 재생성 실패 시에도 이전보다 나은 응답이면 채택
- DRF strip 후 300자 미만 시 검증된 법조문만으로 마지막 재생성 시도
