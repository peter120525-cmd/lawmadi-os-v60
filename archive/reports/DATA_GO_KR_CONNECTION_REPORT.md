#### 발생한 문제
- **HTTP 500 Internal Server Error** 모든 엔드포인트에서 발생
- 응답 타입: `text/html` (에러 페이지)
- 다양한 파라미터 조합(대소문자, 인코딩/디코딩) 모두 실패

#### 가능한 원인
1. ✅ API 키 발급됨 → ❌ **서비스 활용신청 미승인**
2. API 키가 해당 서비스(15000115)에 등록되지 않음
3. 트래픽 할당이 완료되지 않음
4. API 서비스가 일시적으로 중단됨 (가능성 낮음)
