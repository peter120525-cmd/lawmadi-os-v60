### 3. 5단계 표준 응답 구조 ✅

**모든 시나리오에서 일관된 구조**:

```
1. 요약 (Quick Insight)
   → 상황의 핵심 정리

2. 📚 법률 근거 (Verified Evidence)
   → DRF API 실시간 검증
   → 법령/판례 제시

3. 🕐 시간축 분석 (Timeline Analysis)
   → 사건의 시간적 전개
   → 기한과 절차적 제약

4. 절차 안내 (Action Plan)
   → 구체적 행동 방향
   → 단계별 가이드

5. 🔍 참고 정보 (Additional Context)
   → 보충 설명
   → 주의사항
```

```
   "신호 준수 의무 vs 선진입 주장"

3. 입증 구조

4. 절차 전개
   "경찰 조서 → 목격자 확보 → 보험사 대응"

→ 사용자가 자연스럽게 "다음 단계"를 인지
```
