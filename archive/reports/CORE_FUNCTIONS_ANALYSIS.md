**DRF API 검색 한계**:
- 현재 `search_law_drf(query)` 함수는 단일 키워드 검색

**개선 방안**:
```python

    "배당"
])  # 여러 법령 동시 검색

   3. 명의신탁 무효 주장

B. 세입자의 조치

C. 명의자(사용자)의 조치
   1. 채무 확인
   2. 배당 절차 참여
   3. 잔여 재산 정리
```

### ⚠️ 미흡한 부분

**절대 시간 표현 부족**:
- ❌ "경매 기일까지 2주일 이내"
- ❌ "배당요구는 X월 X일까지"
- ❌ "우선변제권 행사 기한"

**시간 순서의 명확성**:
- 현재: A, B, C로 주체별 구분 ✅
- 부족: "즉시 → 1주일 → 2주일 → 경매일" 같은 **순차적 타임라인** ❌

### 💡 개선안

**개선된 시간 배열 구조**:

```markdown
🕐 시간 순서별 행동 계획

┌─ 즉시 (오늘~3일)
│  1. 경매 진행 상황 확인 (법원 조회)
│  2. 확정일자/전입신고 확인 (세입자)
│  3. 명의신탁 증빙 자료 수집 (아버지)
│
├─ 단기 (1주일 이내)
│  1. 배당요구 신청 (세입자 우선)
│  2. 경매 기일 확인
│  3. 법률 상담 (복합 법률 문제)
│
├─ 중기 (2주~1개월)
│  1. 경매 기일 참석
│  2. 이해관계인 협의
│  3. 배당 순위 확정 대기
│
└─ 장기 (경매 이후)
   1. 배당 절차 참여
   2. 명의신탁 정산
   3. 잔여 채무 처리
```

def search_multiple_laws(keywords: List[str]) -> Dict[str, Any]:
    """여러 법령을 동시에 검색하여 통합"""
    results = {}
    for keyword in keywords:
        result = search_law_drf(keyword)
        if result.get("result") == "FOUND":
            results[keyword] = result["content"]

    return {
        "result": "FOUND",
        "laws": results,
        "source": "국가법령정보센터 (다중 검색)"
    }
```

def generate_timeline(events: List[Dict]) -> str:
    """시간 순서로 이벤트 배열"""

    # 과거 → 현재 → 미래 구분
    past = [e for e in events if e['date'] < now]
    future = [e for e in events if e['date'] >= now]

    timeline = "🕐 시간축 분석\n\n"
    timeline += "━━━ 과거 사건 ━━━\n"
    for event in sorted(past, key=lambda x: x['date']):
        timeline += f"{event['date']}: {event['description']}\n"

    timeline += "\n━━━ 현재 위치 ━━━\n"
    timeline += f"NOW: {now}\n"

    timeline += "\n━━━ 향후 절차 ━━━\n"
    for event in sorted(future, key=lambda x: x['date']):
        timeline += f"{event['date']}: {event['action']}\n"

    return timeline
```

#### B. 절차 우선순위 정렬

```python
def prioritize_actions(actions: List[Dict]) -> List[Dict]:
    """행동을 시간 우선순위로 정렬"""

    priority_order = {
        "즉시": 1,
        "긴급": 2,
        "1주일": 3,
        "2주일": 4,
        "1개월": 5,
        "사후": 6
    }

    return sorted(actions, key=lambda x: priority_order.get(x['timing'], 99))
```
