# 🚨 긴급 수리 가이드: 법령 API 연결

**문제**: 법령 검색/조회 기능 전체 중단
**원인**: law.go.kr, data.go.kr 양쪽 API 모두 미설정/미승인
**생성일**: 2026-02-12

---

## ⚡ 빠른 해결 방법 (10분)

### 1단계: law.go.kr 인증키 발급

```bash
# 1. 브라우저에서 접속
https://open.law.go.kr/

# 2. 회원가입 또는 로그인

# 3. 상단 메뉴: [Open API] → [인증키 신청]

# 4. 신청 정보 입력
   - 활용 분야: 법률 상담 서비스
   - 시스템명: Lawmadi OS
   - 예상 트래픽: 1,000건/일

# 5. 신청 완료 → 마이페이지에서 인증키 확인
```

### 2단계: 환경변수 설정

```bash
cd /workspaces/lawmadi-os-v50

# .env 파일 편집
nano .env

# 다음 줄 추가:
LAWGO_DRF_OC="여기에_발급받은_인증키_입력"

# 저장 후 종료 (Ctrl+O, Enter, Ctrl+X)

# 환경변수 로드
source .env

# 확인
echo $LAWGO_DRF_OC
```

### 3단계: 연결 테스트

```bash
# 테스트 스크립트 실행
python test_law_go_kr_eflaw.py

# 성공 시 출력:
# ✅ law.go.kr API가 정상 작동합니다!
```

## 📋 대체 방법: data.go.kr 활용신청

law.go.kr 인증키 발급이 지연되면:

### 1단계: 활용신청 확인

```bash
# 1. 공공데이터포털 로그인
https://www.data.go.kr/

# 2. 마이페이지 → [활용 API 목록]

# 3. "법제처 국가법령정보 공유서비스" 검색
   - 서비스 코드: 15000115
   - URL: https://www.data.go.kr/data/15000115/openapi.do
```

### 2단계: 상태 확인

- **승인 완료**: 1-2시간 후 재테스트
- **미신청**: "활용신청" 버튼 클릭
- **반려**: 사유 확인 후 재신청

## 🔧 트러블슈팅

