### v50 → v60 업그레이드

**하위 호환성:** ✅ **완전 보장**

- v50 API 호출 → v60에서 100% 동일하게 작동
- 기존 프론트엔드 코드 변경 불필요
- 환경변수 그대로 사용 가능

**추가 작업:**
1. DB 마이그레이션 실행 (선택적)
   ```bash
   python scripts/run_migration.py
   ```

2. 새 패키지 설치
   ```bash
   pip install PyPDF2==3.0.1 python-multipart==0.0.9
   ```

3. uploads/ 폴더 생성
   ```bash
   mkdir -p uploads
   ```

**주의사항:**
- 기존 v50 사용자는 아무 영향 없음
- v60 신규 기능은 선택적 사용
